Uma solução para
encher de otimismo.

Bluecup® é uma linha de papelcartão para copos descartáveis feitos a partir de matéria-prima de fonte renovável e 100% produzida no Brasil. Feito com fibras virgens branqueadas de eucalipto, certificadas pela FSC, une inteligência sustentável com tecnologia, oferecendo ao mercado um produto com alta colagem para acondicionamento de líquidos quentes ou frios, para carregar diversos alimentos de forma segura.

Um produto que nasce com um propósito claro: oferecer uma solução de qualidade e contribuir com uma transformação positiva no uso de copos descartáveis.
